# EMD Frontend - React Dashboard

Environmental Monitoring Dashboard built with React, Plotly.js, and Material-UI.

## Features

- **Real-time Monitoring**: Live sensor data visualization
- **Predictive Analytics**: ML-based predictions for temperature, humidity, and gas levels
- **Analysis Reports**: Comprehensive reports with user-specified time periods
- **Anomaly Detection**: Automatic detection of abnormal sensor readings
- **Alert System**: Custom threshold configuration and alerts
- **Monotone Design**: Professional grayscale UI with colors only in charts

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

The app will be available at http://localhost:3000

## Build

```bash
npm run build
```

## Design

- **UI Colors**: Monotone grayscale/black-white
- **Chart Colors**: Full color spectrum for data visualization
- **Icons**: Custom SVG icons (lucide-react) - no emojis

## Technology Stack

- React 18
- Plotly.js for charts
- Material-UI for components
- Axios for API calls
- React Router for navigation
- Vite for build tool
